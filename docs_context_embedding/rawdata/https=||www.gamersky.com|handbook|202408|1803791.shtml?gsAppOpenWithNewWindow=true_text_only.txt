《黑神话悟空》中有着一些难点与疑问困扰着许多玩家，我们针对一些主要的问题做出了解答，下面是由“Shy夏夏”为大家带来的《黑神话悟空》新手难点问题汇总，希望可以帮助到大家。 
Q：怎么洗点？ 
A：在【土地庙（篝火）】可以免费、无限洗点。 
Q：预购、豪华版奖励怎么领取？没奖励会有影响吗？ 
A：第一次见到【土地庙（篝火）】可以直接领取，所有的奖励强度都不会。 
Q：迷路怎么办？

Q：会错过支线或者收集品吗？ 
A：前三章是没有错过就不能捡的内容，所以大家完全可以先自己探索，后续再看攻略进行内容补全（目前已知，如果第三章队友离队后，将无法触发蛇骨+金色蝙蝠的剧情，是否BUG待定……） 
Q：前期怎么加点？ 
A：前期选择一些机制比较好的技能先点上，然后熟悉一下操作，再选择适合自己的流派，多试试，没准会发现新大陆。 
Q：如何跳过动画？ 
A：手柄直接长按X键，键鼠长按E键。 
Q：怎么选择道具？ 
A：手柄直接按十字键，键鼠需要用鼠标滚轮。 
Q：怎么变身？ 
A：随着流程进行，会解锁一种长变身，会完全换一个角色操作，一种是短变身，用于控场和打伤害。 
Q：有弹反吗？ 
A：基本设定没有，但是有一个类似弹反的技能，如果走这个流派的话基本上就可以当做弹反使用。 
Q：武器装备如何选择？ 
A：前期跟着数值走，有什么用什么；后期根据自己常用的流派以及对方的属性进行选择克制方法。 
Q：武器装备能升级吗？ 
A：武器可以锻造成各种形态；装备可以升级，绿→蓝→紫→金→红，所有装备都可以强化到最高级。 
Q：法宝是干什么的？ 
A：其实主要用来打BOSS，其他时候用处并不明显，聊胜于无吧。 
Q：魂魄是干什么的？ 
A：短变身，用来控场和补伤害的。 
Q：BOSS打不过怎么办？ 
A：如果不是最终BOSS，可以刷一下等级和灵蕴（钱），这样可以获得更多的技能以及消耗品，这样可以降低已经难度。如果还是打不过可以查看我们游民星空攻略集的全BOSS打法。

血瓶 → 血瓶附加词条和可镶嵌宝石栏位 → 血瓶镶嵌的宝石
