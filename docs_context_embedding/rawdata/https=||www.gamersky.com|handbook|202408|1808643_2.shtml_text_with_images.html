Title: 《黑神话悟空》全青蛙BOSS位置及打法 波浪浪位置在哪_第二回头目-浪里个波-游民星空 GamerSky.com 第2页：第二回头目-浪里个波 展开 第二回头目-浪里个波 第二回游记词条位置位于“虎先锋”的下方。 浪里个波出现在第二回土地庙“村口”旁边的水池里，外观基本就是换色的第一回波里个浪，攻击模组也有很多一样的。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 但浪里个波十分擅长使用远程雷属性法术，释放频率高释放数量多，比第一回的广谋要难缠很多。 注意观察浪里个波从嘴里吐出的雷球，根据吐出的方式不同其效果也会发生变化，如果是直接朝上吐出来的雷球，会直接砸向周围地面产生一次大范围的AOE雷暴，离远点就能躲过。 如果雷球是从嘴里依次飘出来的则是带有追踪玩家功能的飞弹雷球，分为多个波次攻击玩家，也就是至少需要躲避数次才会结束。 喊叫一声开始往地下吐出的雷球虽不具备追踪玩家的功能，雷球会从超四周衍生出去，造成远距离范围伤害，需要连续跳跃躲避。 除了上述的远程攻击外，还需要额外注意浪里个波的舌头，舌头有两种攻击方式，其一是远距离横扫，其二是一个投技，被舌头碰到的玩家会被强制吞进浪里个波的肚子里造成大量伤害。 因为波里个浪的攻击方式很多且威胁极大，建议开局不要留技能，走到波里个浪旁边先打一套普攻连招再立刻使用定身术继续打一套，把体力打空直接变广智搓伤害，不出失误的情况下这套连招先打半管血没问题。 而我们自身的状态与葫芦都是满的，而通过上述的招式可以看出来波里个浪属于高攻击欲望且拥有多种攻击方式的BOSS，使用的法术攻击都非常克制翻滚穿插普攻的打法，因此建议以高躲避少普攻的方式积攒棍势，找准一个招式后摇直接重棍敲上去，继续等待定身术以开始下一个连招，因为前面已经消耗了半管血，中间插几次重棍再使用第二个定身术收下蛙头。 1 2 3 4 5 6 0 0 文章内容导航 第1页：第一回头目-波里个浪 第2页：第二回头目-浪里个波 第3页：第三回头目-浪里个浪 第4页：第四回头目-波里个波 第5页：第五回头目-波浪浪 第6页：第六回头目-浪波波
