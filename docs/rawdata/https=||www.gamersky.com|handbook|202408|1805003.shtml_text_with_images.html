Title: 《黑神话悟空》石中人位置及任务攻略 石中人商店解锁 2024-08-21 18:03:45 来源：游民星空[原创] 作者：瑞破受气包 我要投稿 第1页： 展开 《黑神话悟空》中的黄袍员外支线需要“醒酒石”，醒酒石可以通过拾取或在石中人商店购买获得，那么石中人商店该如何开启呢？下面就为大家带来《黑神话悟空》石中人位置及任务攻略，希望对大家有所帮助。 石中人位置 石中人位于第二回【黄风岭-挟魂崖-藏风山凹】，在土地庙选择缩地时可以看到这里有NPC标志。 传送到土地庙之后，往右边的大区域走，进入下层通道中的门（参考下图）。 进门之后沿路走到头就能看到石中人，互动他会说他被某些坏人封印在了这里，希望我们帮他解开封印。 对话到重复后，返回土地庙，往左进门往前走。 进门之后可以看到前方有一个巨大的山洞，进门往右走。 往右转之后往左走。注意如果是第一次来的话，前方可能会遇到冒蓝光的精英小怪石双双，建议在此时击败她并吸取精魄，因为后续击杀石母后将无法再获得石双双的精魄。 前方有一条往上走的楼梯，和一个通往室外（枕石坪）的路，这里我们走楼梯往上。 然后沿着唯一的路走到头就能看到BOSS【石母】，击败石母，获得【石之精魄】。 回去找石头人，发现被石头人骗了，愤怒击败石头人，开启石头人商店。 虽然在这里开了商店，但是不建议在这里购买任何东西，石头人是个妥妥的奸商，不仅所有东西卖的特别贵，而且还会出售金稳兽，金稳兽本身就是一种换金道具，买完再卖立减50%。 石中人的商店里还会出售珍玩和泡酒物，暂时没有发现其他的获取途径，可以考虑购买，效果如下。 泡酒物【铁弹】效果：饮酒时，不会被对手打断，且获得一定伤害减免。 珍玩【金花玉萼】效果：些微增加打杀对手获得的灵蕴 除此之外，石中人的商店中还会出售黄袍员外支线任务的关键道具【醒酒石】，这是开启第二回隐藏地图【隐·斯哈里国】的重要道具。 不建议在此处购买，非常浪费钱，因为在后面到达【定风庄】时是可以直接拾取的。但是如果想要留一个作纪念的话，买了未尝不可，完成黄袍员外支线后也可以购买，但目前没有发现其他用途。 醒酒石拾取位置 从第二回【定风桥】土地庙出发，往左往下走过桥 进入村子。 进村后右转走唯一的路从缝隙里穿过去。 前方可以看到三层围墙和一个楼梯，上楼梯后直走往右看，可以看到一个小隔间里有个发光的罐子，击碎罐子即可获得醒酒石。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
