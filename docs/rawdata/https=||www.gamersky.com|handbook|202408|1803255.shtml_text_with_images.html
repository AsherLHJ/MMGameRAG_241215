Title: 《黑神话悟空》第三回泡酒物收集攻略 第三章酒食怎么获得-游民星空 GamerSky.com 第1页： 展开 《黑神话悟空》中的葫芦相当于血瓶，葫芦中可以装不同效果的酒，酒里泡的酒食不同，获得的增益也会有所区别。下面就为大家带来《黑神话悟空》第三回泡酒物收集攻略，希望对大家有所帮助。 其他章节泡酒物（酒食）位置请参考 这篇攻略 。 酒食-特品-十二重楼胶 获取方法：在第三回【浮屠界-安身寺】到达龟将背上的土地庙【龟岛】后，面对土地庙，往龟背右侧下层走，击败最下层的BOSS青背龙（它在龟背的边沿处钓鱼）后，开启青背龙旁边的宝箱即可获得。具体位置请参考下文。 首先从【小西天-苦海-龟岛】土地庙出发，面对土地庙，往右一直走。 走到两块大石头之间发现可以看到下层的区域，青背龙就坐在下层边缘处的石墩上钓鱼。击败青背龙后开启它旁边的宝箱即可获得。 酒食-上品-瑶池莲子 获取方法：在第三回【浮屠界-安身寺】跟随龟将来到苦海后，从土地庙【苦海-苦海北岸】出发，面对土地庙往左沿着海岸走，与辰龙对话并战斗，解锁【如意画轴】后，从六六村村口申猴处购买。 酒食-良品-霹雳角 获取方法：在第三回【浮屠界-安身寺】跟随龟将来到苦海后，从土地庙【苦海-苦海北岸】出发，面对土地庙往左沿着海岸走，与辰龙对话并战斗，解锁【如意画轴】后，从六六村村口申猴处购买。 酒食-特品-龟泪 获取方法：在第三回【浮屠界-安身寺】跟随龟将来到苦海后，从土地庙【苦海-苦海北岸】出发，面对土地庙往右沿着海岸走，可以看到巨大的蛇骨，沿着蛇骨走到尽头可以看到蛇头骨，此时如果猪八戒在队伍中，触摸蛇头骨会触发与无量蝠的BOSS战（猪八戒不在队伍中则不会触发战斗）；击败无量蝠后触发剧情，返回土地庙旁边龟将眼泪掉落的位置即可拾取【龟泪】（目前版本如果拾取龟泪时猪八戒不在队伍中，则游记中的龟将词条不会解锁）。 酒食-上品-铁骨银参 获取方法：采集任意山参采集物都有概率获得。 酒食-良品-甜雪 获取方法 从第三回【小西天-小雷音寺-寺门】出发，面对土地庙往右侧走到头，进入走廊，从走廊沿着唯一的斜坡往上走，然后右转上楼梯。 上楼梯之后，可以看到前方建筑之间有一个巨大的香炉，往香炉的方向走。 香炉后方有扇开启的门，门后是个栏杆，从门后的栏杆处往下跳到下层。 从栏杆跳下去之后，落地往右看可以看到一个亭子，开启亭子里桌子上的宝箱即可获得。 酒食-特品-困龙须 获取方法：在第三回【雪山径-照鉴湖】往湖边跑，然后往右跑到最远处，可以看到一个桌子上有宝箱，开启宝箱获得。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
