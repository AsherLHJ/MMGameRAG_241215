Title: 《黑神话悟空》全道具作用说明 各道具功能一览 2024-08-20 10:08:05 来源：游民星空[原创] 作者：Shy夏夏 我要投稿 第1页： 展开 《黑神话悟空》中有着许多道具，许多小伙伴想知道各个道具都有什么具体功能，下面是由“Shy夏夏”为大家带来的《黑神话悟空》全道具作用说明，希望可以帮助到大家。 武器 除了初始武器，所有武器都是需要进行打造的，同一类武器进行不同的锻造会返还之前锻造的材料，但是不会返还之前锻造消耗的灵蕴。 前期（一至三回）建议武器选择属性更高的，再第三回后半或者后续关卡，更建议使用和自己常用流派契合的武器进行战斗。 法宝 前两回的最终BOSS如果都是难度较高的，而使用隐藏地图中获取的法宝可以很好的克制BOSS，更轻松地战胜他们。 精魄 首先要看它的携带效果，绝大多数情况精魄主动技能都是用于控场或者追加伤害的，所以不用太在意，只要看他的被动效果是否和自己的流派更搭即可。 防具 前期（一至三回）建议防具选择属性更高的，，再第三回后半或者后续关卡，更建议使用和自己常用流派契合，以及克制对手属性等方面进行选择。 珍玩 初始只有2个栏位，解锁家园后，战胜寅虎可以获得第三个栏位。 葫芦 可以理解为血瓶，不同血瓶的使用次数是不一样的，效果也不同。 酒 可以理解为血瓶附魔，给血瓶额外的效果。 泡酒物 可以理解为血瓶附魔再镶宝石，不同附魔可以镶嵌不同数量的宝石效果。 用品-缩地神符 因为本作死亡和传送都没有惩罚，所以用的时候不多，主要跑图时使用。 用品-丹药 所有效果写着【一定时间】为【60秒】，【较长时间】约为75秒，【长时间】为【130秒】。 材料-兵器专材 一般这种材料都会解锁武器的新配方，但依然收集材料需要手动锻造。 材料-兵器通材、披挂同材 用于锻造和强化武器装备的，按照【绿＜蓝＜紫＜金＜红】的稀有度，会随着剧情逐步解锁，如果没有获得，推一推剧情，会逐步获得。 稀罕材料-小活师 强化青蛙精魄使用。 稀罕材料-玲珑内丹 强化主角的各项属性，可以刷取。 稀罕材料-铁中血 强化蓝色精魄用的。 稀罕材料-虚空结 强化紫色精魄用的。 稀罕材料-落伽香藤 增加血瓶的使用次数。 稀罕材料-三冬虫 提升酒的效果。 药材配方与药材 合成丹药用的，配方和药材都是在野外采集获得，有一定随机性，大家见到野外的植物最好都捡了，不要觉得浪费时间。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
