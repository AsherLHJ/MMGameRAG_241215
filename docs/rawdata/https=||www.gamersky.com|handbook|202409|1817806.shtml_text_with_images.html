Title: 《黑神话悟空》四妹人物剧情解析 2024-09-18 10:54:21 来源：黑神话吧 作者：哦(﹃ ) 我要投稿 第1页： 展开 《黑神话悟空》中的四妹是最让玩家意难平的一名角色，并且游戏中四妹的剧情有很多留白，很多玩家通关后对她的剧情也不是很理解。下面为大家带来由“哦(﹃ )”分享的《黑神话悟空》四妹人物剧情解析，一起来看看吧。 1.村口初见：和二姐作为战力前二目标就是来抓八戒的，四妹负责在村口盯梢顺便清杂二姐负责在村里埋伏八戒，不过杀那两个小啰啰单纯是他们说蜘蛛坏话，按蜘蛛的原计划估计是四妹打天命人。然后八戒变身的消息应该就是四妹告诉二姐的。 2.落蜘蛛洞：个人感觉在村中到落洞之间最初应该是安排了个四妹boss战（应该是打的人形态，最后变蜘蛛把猴控到洞里网上接剧情）。 3.洞内：这里应该是直接砍了一大段的像八戒那样入队或狐狸那样的导游（这也是为啥现在剧情四妹总是突然跳出来谜语人感觉跟跟踪狂一样）。四妹没按计划砍了猴以帮我们捞八戒为交换让我们帮忙解封印。 4.虫卵：嫁妆那块应该是老蜘蛛收的贺礼，里面的虫子才是重点。原来天庭炼丹应该是主线，我们和四妹在路上逐渐了解真相，这也是四妹开始流泪的原因，同时应该是这个时候才搞清楚封印是紫符（不然之前就知道肯定试过明知撕不了还要当着我们面再撕确实有点茶）。 5.搭桥：四妹肯定不能直接帮忙揍蜘蛛的，所以这里离队。 6.和二姐对喷：说有了好东西先让姐妹尝尝，虽然这大概率是因为知道二姐参与交易不爽在顶她随便给猴子拖时间。但她要真是未亡人那上次结婚的时候给姐妹们分享了？喷人不能用自己的软肋啊。（而且应该是在剧情中有四妹和家人决裂的戏份，当然也是变相的保护家人。不然最后全家少个人没有一个人问？并且应该是二姐后来和大姐谈过，所以最后百眼她没低头，同时上的时候两次看大姐。） 7.山上：这里应该是又入队了然后砍了很大一段剧情。应该是有一段一起战斗捣毁生意的boss战（蜘蛛更多是打工人而不是炼丹原料，不然做这么多年生意了姐妹们还整整齐齐，四妹更多出于正义我感觉形象才更加具体） 8.炼丹：接上面说的，天庭原来的计划不是蜘蛛炼丹，四妹炼丹单纯是惩罚手段。而且老鹤上来就说人情，四妹要真是天命猴夫人，搅你个生意还不是我猴哥一句话的事（不过按这个剧情可能真炼丹了，为了苍生舍身取义了） 总结 我个人还是倾向于打造四妹勇敢、正义及心系苍生的形象，而不是把行为简单的归结于轮回的爱情。 当然，从目前的反响来看，后期加剧情甚至改剧情的可能性还是很大的。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
