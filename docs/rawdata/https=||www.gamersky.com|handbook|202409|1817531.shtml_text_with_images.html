Title: 《黑神话悟空》召唤型精魄性能测试 召唤型精魄伤害对比 2024-09-17 10:49:33 来源：黑神话吧 作者：Wrathion2020 我要投稿 第1页： 展开 《黑神话悟空》中共有4种召唤型精魄，许多小伙伴想知道哪个比较厉害，下面是由“Wrathion2020”为大家带来的《黑神话悟空》召唤型精魄性能测试，希望可以帮助到大家。 实战性能外指标 1.火灵元母，元气消耗11，召唤4只小火蛇，持续约16秒 2.老人参精，元气消耗11，召唤1只大人参精，持续约32秒 3.广谋，元气消耗8.5，召唤4只小毒蛇，持续约10秒 4.儡蛛士，元气消耗11，召唤6只小毒蜘蛛，持续约16秒 测试方法 7周目第四周金光苑，门口清理两个小怪后，隐身到蛇捕头面前然后放召唤精魄，玩家本人尽可能不参与战斗，也完全不造成任何伤害。 评判标准 在召唤物全部死亡或持续时间结束后，蛇捕头损失的血量为评判标准，对于有异常dot伤害的精魄，以等待至dot伤害全部结算后的血量为评判标准。 测试结果 火灵元母 战绩：1-4 评价：仅1次打死蛇捕头，其余4次大约打了85%～90%血量，伤害中等，攻击频率中等，削韧低，综合性能还不错。 老人参精 战绩：1-4 评价：作为唯一一个单挑的召唤物，上下限差距比较大，同样仅一次获胜，其余时候大概能打掉60%～90%血，因为自身韧性比较低，攻击出手慢，所以常常被打断或者被后撤躲开，不过攻击偶尔能打断蛇捕头，因此运气好还是能获胜的，单次伤害高，攻击频率很低，削韧中等，综合性能一般。 广谋 战绩：0-5 评价：前期就能拿到的精魄，虽然看战绩最差，但广谋消耗比其他三个都低，能稳定打掉蛇捕头70%左右血，这个伤害能力已经超过大部分同消耗的攻击类精魄，唯一缺点是持续仅10秒，因此只能挂一次毒，略有可惜，攻击频率高，伤害能力较低，削韧低，综合能力还不错。 儡蛛士 战绩：5-0 评价：黑马！召唤类精魄中唯一一个仅蓝色稀有度的，但实战表现却最亮眼，召唤整整6只带毒小蜘蛛，小蜘蛛很灵活，攻击有自适应位移，能直接扑到敌人面前进行攻击，攻击能上毒，并且攻速快，配合数量多，大部分时候能把蛇捕头怼在墙角动弹不得，且小蜘蛛死亡或持续时间结束都会自爆，造成伤害和毒异常，但测试时4次都是全员存活战胜蛇捕头，仅1次死了1只，因此自爆的优势几乎完全没有发挥，蛇捕头对小蜘蛛来说还是太弱了，8900血是蛇捕头的上限，不是小蜘蛛的上限，攻击频率高，伤害能力高，削韧高，综合能力非常高，喜欢玩召唤流的可以尝试，儡蛛士被动还能减少身外身法20点蓝耗。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
