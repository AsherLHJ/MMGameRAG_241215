Title: 《黑神话悟空》易错过内容及不可逆支线任务汇总-游民星空 GamerSky.com 第1页： 展开 《黑神话悟空》中有一些道具物品只有在特定的剧情节点才能拿到，还有一些支线任务在触发特定主线后就会错过，下面就为大家带来《黑神话悟空》易错过内容及不可逆支线任务汇总，希望能帮助各位小伙伴成功避坑。 ※本文不可避免涉及剧透（包括主线/支线剧情、BOSS、隐藏区域、结局），请大家酌情阅读。 ※在触发结局后开启新周目之前（再入轮回），除去下文提到的一些不可逆内容之外，都可以在标题界面选择【继续游戏】去继续完成，一定要确保没有遗漏再开启新周目。 第一回 幽魂精魄 第一回【黑风山-苍狼林-林外】的BOSS幽魂，如果在击杀幽魂之前先去敲钟击败了位于隐·旧观音禅院的【金池长老】的话，【幽魂】就会消失，导致无法获得精魄，暂时没有办法补救，错过的话只能通过二周目收集。 注意在第二回可以获得的四渎龙神的支线必备道具【飞龙鳞片】，如果在与BOSS沙大郎战斗期间没有撞开墙的话，可以使用其他手段撞破墙，例如幽魂精魄/变化碧尘/四段劈棍蓄力等。虽然幽魂精魄没有获得不会影响【飞龙鳞片】的获取（参考下文第二回），但开门手段中的幽魂精魄是最容易使用的，尽量不要错过。 幽魂精魄的详细获取方法请参考 这篇攻略 。 隐·旧观音禅院进入方法请参考 这篇攻略 。 马天霸支线任务 马天霸的支线会在第一回至第五回出现，其中第一回至第四回，每一回都需要找到马天霸并与其对话至重复，才能不漏掉他的支线任务的每一个步骤，确保在前四回没有漏掉他的对话后，在第五回火焰山再次与他对话，击败BOSS火焰山土地后，返回与他对话即可获得法术【黯雷】。 注意只要有任意一回漏掉对话，在完成第四回（击败百眼魔君）之前依然可以返回前几章的指定位置与他对话到重复，目前测试进入第五章后如果之前有漏掉的对话，将无法完成马天霸支线的最后一步，导致无法获得法术【黯雷】。建议在每次推进主线剧情进入下一章之前，都尽量与该章节的马天霸对话至重复后再进入下一章，以免错过。 马天霸的完整支线任务请参考 这篇攻略 。 第二回 沙二郎精魄 第二回【黄风岭-沙门村-不济谷】的BOSS沙国王&沙二郎，需要先击杀沙国王再击杀沙二郎，并且在沙二郎进入狂暴模式之后再击杀沙二郎，就能获得沙二郎的精魄。如果在战斗期间先击杀沙二郎再击杀沙国王，会导致漏掉沙二郎的精魄，暂时没有办法补救，错过的话只能通过二周目收集。 顺带一提，在【沙国王&沙二郎】战斗期间，击杀沙国王和沙二郎的先后顺序会影响到后续灵吉菩萨的唱词，以及在BOSS沙大郎处触发的具体剧情（沙国王活着被沙大郎吃掉或用沙国王的肉引出沙大郎），注意这一点只会影响沙二郎的精魄和沙大郎剧情，不会影响沙大郎的BOSS战触发。 注意在与BOSS沙国王&沙二郎战斗期间，沙国王会站在场地中间的四根石柱上，诱导沙二郎击碎石柱，或者蓄力重击/幽魂精魄/碧尘变化击碎石柱后，沙国王就会掉落到地面上。 沙二郎精魄的详细获取方法请参考 这篇攻略 。 石双双精魄 第二回【黄风岭-挟魂崖-藏风山凹】的石双双，如果提前击杀了【石母】的话，可能会无法获得【石双双】的精魄，暂时还没有找到补救方法，错过的话只能通过二周目收集。 石双双精魄的详细获取方法请参考 这篇攻略 。 鼠都尉、沙国三王子、小张太子支线任务 小张太子（沙国三王子）的位置位于第三回，但他的支线任务在第二回就已经有前置了。在第二回黄风岭-沙门村侧面通往BOSS沙国王&沙二郎所在区域的大门前，可以看到两只双头鼠，大门斜对面有个小入口通往BOSS地狼所在区域，在小入口旁边有个卡在石头里的木房子，靠近可以互动与里面的鼠都尉对话。 在第二回如果没有与鼠都尉对话至重复，或者没有帮助鼠都尉击杀2只双头鼠，那么鼠都尉就会出现在第三回【小西天-浮屠界】中的下层和上层土地庙之间那层的尽头（第二层尽头的牢房里），击败BOSS鼠都尉，即可获得法术【灰蜇】。 如果在第二回听完了鼠都尉的对话，并且击杀了2只双头鼠，那么鼠都尉的尸体就会出现在第三回【小西天-浮屠界-下层】天命人初始位置隔壁的牢房中（牢房门上有紫色符咒，需要击败上层土地庙旁边坑里的魔将·莲眼才能打开），与小张太子在同一间牢房，拾取尸体上的武器即可获得法术【灰蜇】。这种情况下，在监牢第二层尽头是不会触发与鼠都尉的BOSS战的，但不会漏掉任何法术和道具。 小张太子支线的详细步骤请参考 这篇攻略 。 醒酒石 醒酒石是黄袍员外支线任务的必要道具（涉及到开启【隐·斯哈利国】），每一个周目最多都可以获得2块，有一块是在地图上拾取的，还有一块需要通过石中人支线开启石中人商店购买获得。醒酒石只需要1块就能完成任务，另一块没有其他用途。但在石中人商店购买价格会很高，前期负担比较大，建议通过地图拾取，不要在商店购买。 如果先开启了石中人商店并在商店中购买了醒酒石，则地图拾取的那一块就会消失，无法前往拾取。 如果先通过地图拾取了醒酒石，再通过石中人商店购买，就能获得2块醒酒石（虽然只需要1块就能完成任务，剩下一块没有其他用途）。 醒酒石的详细获取方法请参考 这篇攻略 。 黄袍员外支线任务（开启隐·斯哈里国）详细步骤请参考 这篇攻略 。 石敢当&石先锋对战 如果在收集6个佛目珠并唤醒石敢当之前，先击败了BOSS石先锋，就会错过石敢当和石先锋对战的场景，但不会漏掉相关道具。想要自己挑战石敢当和石先锋的话，可以先打石先锋，击败石先锋之后再集齐6个佛目珠召唤石敢当，这样就能分别挑战两个BOSS了。 6个佛目珠的位置请参考 这篇攻略 。 飞龙鳞片、龙珠、天龙棍 飞龙鳞片是四渎龙神支线的必要道具，拿到之后才能在第一回、第二回、第三回和第四回找到四条龙（赤髯龙、小骊龙、青背龙和小黄龙），击败获得4个龙珠，除此之外在第三回【小西天-苦海-龟岛】击杀BOSS亢龙星君也可以获得1个龙珠。也就是说一个周目最多可以获得5个龙珠。 制作神珍武器【天龙棍】需要9个龙珠，也就是说至少需要两个周目才能制作，建议第一个周目不要漏掉龙珠，否则不够9个龙珠将无法制作武器天龙棍。 飞龙鳞片获取方法请参考 这篇攻略 。 四渎龙神的支线任务请参考 这篇攻略 。 武器天龙棍的制作方法请参考 这篇攻略 。 第三回 掌灯狱使、珍玩吉祥灯 掌灯狱使一共有9只，分布在第三回【小西天-浮屠界】的土地庙下层（第一层）-上层（第三层）-轮藏（顶层），只有在进入红眼状态时才能看到，普通状态下是看不到的。如果在击杀BOSS【魔将·妙音】之前没有击杀所有的掌灯狱使并拿到珍玩【吉祥灯】的话，在击杀魔将·妙音之后将无法进入红眼状态，导致无法再获得这个珍玩，暂时还没有找到补救方法，错过的话只能通过二周目收集。 吉祥灯的详细获取方法请参考 这篇攻略 。 无量蝠的影神图图鉴 无量蝠位于影神图-游记-头目中，是第三回的头目，必须在首次到达苦海北岸八戒在队伍中时与其战斗并将其击败才能解锁此条目，在八戒离队后将无法解锁此条目，暂时没有补救办法，错过的话只能通过二周目解锁。 同时建议在首次到达苦海北岸并在第一个土地庙上香之后（八戒必定在队伍中）就尽早去触发与无量蝠的BOSS战，因为无量蝠在进入极乐谷之后就会前往小雷音寺（滑雪时会触发无量蝠飞往小雷音寺的动画），不会再在苦海出现。 无量蝠的具体位置请参考 这篇攻略 。 灵狐支线、白狐毫、不空 在第三回【小西天-极乐谷-快活林】土地庙旁边的大门旁与尸体上的狐皮对话，接取支线获得变狐狸的能力，受击时会变为原型（也可以手动变回原形），打开行囊再次点击白狐毫可以随时变化。注意这里在进入小雷音寺的偏殿打BOSS不空之前，一定要变为狐狸再跑进不空所在的房间，否则目前测试在某些版本中可能会导致此支线中断，无法提交任务。 打完不空之后，可以返回快活林接任务的地方交任务，交完之后获得珍玩【白狐毫】，但会失去变成狐狸的能力，珍玩和能力同一周目不可兼得，如果想要变成狐狸快速跑图的话就不要交任务，想要拿到珍玩的话，就一定要去交任务。 珍玩白狐毫的详细获取方法请参考 这篇攻略 。 如意画轴、披挂升阶 在第三回【小西天-苦海-苦海北岸】完成辰龙支线解锁如意画轴之后，即可进入如意画轴的六六村，在六六村村口找到寅虎可以进行披挂升阶，披挂只能升阶不能降阶段，有些披挂升阶之后外观会变化，就算不喜欢升阶后的外观，也不能变成之前的外观了，建议大家参考下文的套装图鉴来查看满级后的效果和外观再优先升级自己喜欢的披挂。除此之外，披挂升阶会消耗一些稀有材料，有些材料的获取在一个周目内是有限的，也建议大家按需升阶。顺带一提，二周目之后披挂升阶可以在土地庙直接进行，无需进入如意画轴。 全套装外观及满级属性请参考 这篇攻略 。 如意画轴的解锁方法请参考 这篇攻略 。 第四回 紫色符咒、四姐支线 紫色符咒是四姐支线，需要把所有符咒都揭下才能完成支线。紫色符咒总共有4张，有3张在盘丝洞内，1张在黄花观，在盘丝洞内的3张没有固定的揭符顺序，只有在花间桥附近的那张才会触发揭符遇到四姐的剧情，剧情是无论揭符先后顺序都不会错过的。 重点在于第四张紫色符咒，第四张紫色符咒位于黄花观金光苑，需要在击败BOSS百眼魔君之前揭开才能完成支线，击败百眼魔君后将无法揭开第四张符咒，导致支线无法完成，并且导致无法解锁影神图人物图鉴【鹤仙人】。 目前测试在触发百眼魔君BOSS战之后强制退出游戏/回档/复活后依然可以前往另一边揭第四张符，但具体时间点尚未测试，建议大家还是在进入百眼魔君BOSS房触发剧情之前把第四张符揭开最保险。 四姐支线详细步骤请参考 这篇攻略 。 黑手道人、臂甲下毒手 在第四回【盘丝岭-盘丝洞-碎玉池】触发与BOSS黑手道人的第一次战斗期间，打掉黑手道人背后的手臂再击杀黑手道人，就能获得他的臂甲【下毒手】。否则在直接击杀黑手道人后将无法获得，暂时没有补救方法，错过的话只能二周目收集。 目前测试最好在第四回【盘丝洞-碎玉池】触发的与【黑手道人】的第一次BOSS战期间获得此臂甲，因为第二次遇到他时他可能不会使用戳地技能，打掉手臂的限制条件会比较多，难度比第一次战斗期间相对要大一些。 黑手道人位置、下毒手的具体获取步骤请参考 这篇攻略 。 毒敌大王、昆蚑毒敌甲、彩玉葫芦 在第四回【盘丝岭-隐·紫云山-绕仙居】可以第一次遇到BOSS毒敌大王，此时靠近会触发对话，击碎毒敌大王所在房子下方的酒坛子即可触发与BOSS毒敌大王的战斗，此处击败毒敌大王可以获得制作昆蚑毒敌甲的必要材料，以及彩玉葫芦。 如果在此时没有击败毒敌大王，直接前往隐·紫云山【巢云顶】触发与BOSS晦月魔君的战斗，那么毒敌大王就会出现在与晦月魔君的战斗中，帮助天命人打掉晦月魔君的第一阶段，同时触发一些对话，无论毒敌大王的血量如何，之后晦月魔君进入二阶段时毒敌大王必死，导致无法获得制作昆蚑毒敌甲的必要材料和彩玉葫芦。 昆蚑毒敌甲的具体获取方法请参考 这篇攻略 。 靡道人 在第四回【盘丝岭-隐·紫云山-落花庄】遇到靡道人后触发靡道人的支线，需要通过击败庄子里的道士获得关键道具【藕雹】，如果没有拿到藕雹或者在拿到藕雹之后没有返回交给靡道人并完成后续支线，直接去打巢云顶的BOSS晦月魔君，靡道人就会消失，靡道人支线中断，导致无法获得法术【藕雹】。 必须要在拿到藕雹交给靡道人，并完成靡道人后续支线之后，再去打晦月魔君，才不会漏掉。 靡道人支线的详细步骤请参考 这篇攻略 。 四渎龙神、小黄龙 小黄龙是四渎龙神的最后一只，如果前面的三条龙（赤髯龙、小骊龙和青背龙）没有全部击败的话，小黄龙是不会出现的，建议根据主线剧情推进进度按照正常跑图流程到达【盘丝岭-盘丝洞-堕龙壁】时就检查飞龙鳞片的获取以及其他三条龙的击杀情况，尽量在通关完整游戏前击杀BOSS小黄龙。因为目前测试在部分版本通关之后通过【继续游戏】返回寻找小黄龙时，小黄龙已经消失。 小黄龙的位置及打法请参考 这篇攻略 。 第五回 兴烘掀·掀烘兴精魄 兴烘掀·掀烘兴是出现在第五回【火焰山-火光地-火燎三关】主路上的大铁球，根据主线推进会从悬崖掉落在【火焰山-丹灶谷-翠云殿】，掉落之后可以拾取精魄，注意这个精魄是不会出现在土地庙拾遗的，只能自己前往掉落地点拾取。 在剧情里大铁球滚落后小狐狸会说【猪八戒皮糙肉厚摔不死的，先去救父王】，实际上这里就已经可以拿到这个精魄了，如果天命人对八戒有一丢丢担心的话，可以直接传送到翠云殿，穿过翠云殿就能在山脚下拾取到这个精魄，但是看不到八戒，后续尚未测试将剧情推进至哪个具体的节点这个精魄会消失，但有些版本已经证实不及时拾取的话确实会消失，所以建议大家还是尽早去拾取，以免漏掉。 精魄的详细获取方法请参考 这篇攻略 。 大力铁角、混铁棍、大力套装 制作神珍武器混铁棍和防具大力套装的必要材料【大力铁角】在同一个周目最多只能获得4个（开启隐·璧水洞宝箱、击败璧水金睛兽），想要全部锻造总共需要消耗8个大力铁角，因此至少需要2个周目才能集齐足够制作所有套装的材料。由于一个周目只能获得4个，并且获取的剧情节点都非常靠后，建议一个也不要漏掉，否则将无法获得足够制作的材料，导致无法在两个周目解锁相关成就/奖杯。
