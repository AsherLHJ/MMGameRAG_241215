Title: 《黑神话悟空》高周目减伤流轮椅配装推荐 2024-09-03 15:46:32 来源：黑神话吧 作者：llllllljh 我要投稿 第1页： 展开 《黑神话悟空》高周目下的敌人难度会有较大地提升，那么如何配装才能让角色更肉一些呢？下面请看由“llllllljh”带来的《黑神话悟空》高周目减伤流轮椅配装推荐，希望对大家有用。 到了高周目，我们天命人的装备就没办法再提升，防御力已经到达了上限，而高周目的怪伤害又高，容错率低了不少，这时候就突显出百分百比减伤的好处。 这里我推荐两套减伤流派，看玩家各自的选择。 第一套-大圣套减伤流 穿上大圣四件套，在三周目打完杨戬之后，打满4豆，再积蓄能量，打出四豆重棍就能激活大圣模式，天命人会说出语音彩蛋，然后招式全部变为新的招式，获得10%加伤以及25%减伤效果。这个效果可以持续1分钟，然后需要再次触发。 带上青田葫芦（二周目申猴购处购买），泡酒物选择3个梭罗琼芽（在沙门村村里的箱子打开获得），珍玩选择博山炉（首次击杀赤髯龙获得），喝一口酒就能获得45%减伤。珍玩带一个琥珀念珠加快攒棍势速度，更高频进入大圣模式 再带上不净精魄，铜头铁臂后增加15%减伤，总计就有85%减伤，但是减伤有上限，最多80%。会浪费掉5%，为什么选择不净而不是地罗刹呢？是因为大圣套的被动，释放法术可以加8%暴击率，而暴击又可以减少法术冷却时间，铜头铁臂点满后只要反震成功就能回蓝，等于不耗蓝。到了后期我们的铜头铁臂就能无限释放。并且不净精魄连砍也比地罗刹好用很多。如果对自己铜头铁臂没有信心的话，可以选择去掉一个梭罗琼芽，选择铜丸，这样最多75%减伤，喝酒之后铜头铁臂不耗蓝，可以反向回蓝，根本不怕不够用。 第二套-牛魔王减伤流 葫芦和大圣套带一样的，法宝换成定风珠，常驻2%减伤，珍玩带上月玉兔和日金乌常驻3%减伤，加上喝酒后45%就有50%，精魄选择地罗刹，和牛魔王套被动很契合，挨打10%减伤就有60%了。虽然减伤比大圣套少，但是牛魔王套有更高的基础防御，带上混铁棍伤害也会更高，并且霸体非常好用，特别适合识破，再加上棍势容易攒，配合识破加点棍势越高伤害越高。而且大圣模式也并不能一直触发，但大圣套的珍玩法宝可选性更多。 大圣套玩法就是要频繁触发大圣模式，多用铜头棍花攒棍势。 而牛魔王套就是纯莽夫，多用识破打伤害和硬直就行。 大圣模式攒的豆变回来也能用，所以感觉快结束了可以攒满4豆，就能快速回到大圣模式，而且大圣模式天命人的棍子更长攻击范围更大。 实测四周目boss一刀只掉血皮，非常肉。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
