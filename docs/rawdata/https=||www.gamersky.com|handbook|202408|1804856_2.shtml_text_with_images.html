Title: 《黑神话悟空》金池长老与赤髯龙位置说明 黑神话悟空第一章隐藏BOSS在哪_赤髥龙-游民星空 GamerSky.com 第2页：赤髥龙 展开 赤髥龙 第二个来介绍如何找到黑风山的隐藏妖王「赤髥龙」，首先要先取得「飞龙鳞片」，「飞龙鳞片」不止是找到「赤髥龙」的条件，也是其他隐藏龙系妖王所需要的道具，关卡要先推进到打通黑风山，并来到第二关「黄风岭」，之后沿着主线道路走，遇到沙国王父子。 这场战斗是能否获得沙二郎精魄的关键，如果照一般剧情，直接打死沙二郎的话，爸爸沙国王会逃跑跑走，必须要先打死沙国王，再打死沙二郎，才能在战斗结束后获得沙二郎的精魄。 想要先打死沙国王，首先要把沙二郎打到半血，这时候沙国王会跳下来加入战斗，再赶紧把他打死即可，注意爸爸死后儿子会爆走，可以预留一点技能、法力值来打，会比较好打。 打完沙国王父子后，来到左边的洞窟，进去洞窟后往左回转，走到一处悬崖前面。 如果是先打死沙二郎，会自动触发剧情，沙大郎会自己出来，如果是先打死沙国王，就要在悬崖处互动，丢出一块血肉召唤沙大郎，这场战斗在打的时候可以注意一下，站位靠近位于悬崖左侧的墙壁，能诱导沙大郎直接把墙壁撞开。 但如果战斗中沙大郎没把墙撞开也没关系，可以使用幽魂的化身技能，一个头槌把墙敲破，或是等级够高的话，使用蓄力4段的棍重击也可以敲破，敲破后从墙的缝隙里面进去，就能在宝箱里面，找到隐藏道具「飞龙鳞片」了。 顺利取得「飞龙鳞片」后，回到黑风山，透过土地庙传送到「苍狼林」的「林外」，往前走在第一个岔路左转，继续沿着路走，会看到一个宝箱。 接着往左边走，就能听到有旁白在讲话，往声音的来源走，能在远处看到一个瀑布。 沿着路走到瀑布旁边，与瀑布互动后，就能进到隐藏妖王「赤髥龙」的BOSS房间了。 打赢「赤髥龙」后，会拿到一个珍玩「博山炉」，以及可以制造许多强力武器的素材，是很值得一打的隐藏妖王哦。 1 2 0 0 文章内容导航 第1页：金池长老 第2页：赤髥龙
