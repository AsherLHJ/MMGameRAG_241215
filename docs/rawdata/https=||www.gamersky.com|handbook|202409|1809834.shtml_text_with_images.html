Title: 《黑神话悟空》二郎神杨戬配装打法心得 二郎神怎么打 2024-09-01 11:00:10 来源：黑神话吧 作者：诗人亦然js 我要投稿 第1页： 展开 《黑神话悟空》中的二郎神杨戬是一个隐藏BOSS，想要解锁真结局就需要与其战斗。下面请看由“诗人亦然js”带来的《黑神话悟空》二郎神杨戬配装打法心得，希望对大家有用。 杨戬这个英雄，想必是很多人在本游戏遇到的最难的一个BOSS了，不仅打他很难，其实找到他更是一件难事（需要做五条隐藏支线，找到五张隐藏地图，打五个隐藏BOSS，打完4条龙，和葫芦仙人在火焰山进行最后一次对话）。 我在这里，写下这篇打杨戬的攻略，希望能帮助到那些正在被杨戬拷打以及那些听闻杨戬难而望而却步的人。 首先，打杨戬，我们需要分析他的特点，他的几大特点我归纳为就是: 读指令的侧身躲避，很容易让你重击落空。 四重甲，不破甲很难打出杨戬的硬直。 大开大合的招式，能很容易就打出你硬直的飞剑，能秒人的天眼和红光。 以及投技。 那么这几点，最好克服的，就是破甲了。 由于我们在火焰山做了主线，所以给了我们芭蕉扇这个法宝。 这个法宝是削韧神器，用一次就能破盾。 但是我们又得知，使用法宝需要一个属性，就是元气，那么，怎么回元气呢。 这个时候，我们就需要去寅虎那里买一套金身装备，能回复元气的四件套。 就算质量是仙品也可以（但是打到这里给的材料，已经够头冠，臂甲，腿甲升到神珍了），因为我们主要是激活这个BUFF:暴击和打杀对手时，获得些微元气。 其中，暴击率的高低就决定了你在攻击杨戬时回元气的速度，所以我们记得带猫睛宝串这个珍玩，以及备好倍力丸或者龙光倍力丸。 这样就能快速回元气。 然后我们的打法思路就是破完盾，用戳棍的搅阵打他，搅阵打人形怪有奇效，能一直压着打，但是首先我们要抓好角度，尽量让芭蕉扇把杨戬给扇到墙角。 可是精力条难免有限，为了能够一直打出搅阵，我们需要备好登仙散。 那么这时候就有人会问了，如果我扇完了破了盾打完一次搅阵，到杨戬的下个形态了，芭蕉扇还没好怎么办。 这个时候我们就要有后手准备，我们需要备好石头人的变身（这个变身削韧相当恐怖，所以破盾效率也是变身中最有效率的）。 需要我们提前把杨戬引到墙角怼他。这样才能让变身的效率最大化。 我们需要备好镜中丹作为备用，作为芭蕉扇和变身都没好时我们的应对措施。 然后打杨戬变身为法天象地的攻略，我们需要知道，金身格挡住可以恢复架势条，而架势条攒够后的重击，可以回血，记住在它开大范围伤害的时候一定要记得格挡，收益相当高。 这就是我想带给大家的杨戬攻略了，如果不想用戳棍搅阵流的，也可以用我这种思路破敌，主要就是用各种方法破盾，压着打杨戬，不论是破完盾后的隐身蓄力重击，又或者是召唤猴子猴孙正义群殴，都不失为一种破敌技巧。 这也是我打了几个小时的杨戬，摸清楚的规律。 游民星空《黑神话悟空》互动地图 现已上线，地图不仅囊括土地庙等地点、收集物品的点位，还标注了全部Boss、隐藏支线和重要道具的点位，照着地图走轻松全收集。 《黑神话：悟空》互动地图 游民星空 《黑神话悟空》原创全收集互动地图 现已上线！全部NPC支线、隐藏点位、重要道具一应俱全。还配有详尽的图文与视频攻略，一站式满足所有收集需求！ 0 0
